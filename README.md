- **Splash Screen**  
  Tampilan awal aplikasi sebelum masuk ke menu utama.  
  ![alt text](https://github.com/BudiAmin/uts_app_mobile/blob/main/images/Splash_screen.jpg?raw=true)

  Menampilkan informasi pribadi dan menyediakan form untuk menambahkan data, namun saat ini data belum tersimpan ke database.  
  ![alt text](https://github.com/BudiAmin/uts_app_mobile/blob/main/images/Biodata.jpg?raw=true)

  Menampilkan 15 daftar kontak, namun data masih berupa dummy.

  Kalkulator sederhana yang mendukung operasi tambah, kurang, kali, bagi, akar, dan kuadrat.  
  ![alt text](https://github.com/BudiAmin/uts_app_mobile/blob/main/images/Kalkulator.jpg?raw=true)

- **Cuaca**  
  Menampilkan informasi cuaca, tetapi data masih dummy.  
  ![alt text](https://github.com/BudiAmin/uts_app_mobile/blob/main/images/Cuaca.jpg?raw=true)

- **Berita**  
  Menampilkan halaman berita dari sumber eksternal (contoh tampilan).

  Menampilkan informasi gempa terkini menggunakan API BMKG.
